Gestion de l’état pour l’intégration avec Tauri.

Ce module, tauri_state, permet de gérer l’état partagé entre les différentes composantes de l’application, en particulier lors de l’utilisation de Tauri pour créer une interface graphique. Module pour gérer l’état de Sonar.

Ce module fournit les structures nécessaires pour maintenir l’état actuel de l’application Sonar, en particulier pour suivre les trames réseau.
