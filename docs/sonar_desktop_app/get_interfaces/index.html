Récupération des interfaces réseau.

Le module get_interfaces permet d’identifier et de lister les interfaces réseau disponibles sur la machine. Il est essentiel pour permettre à l’utilisateur de sélectionner l’interface sur laquelle écouter le trafic réseau.

Functions§

  • get_interfaces
    Récupère les noms de toutes les interfaces réseau sur le système, avec une entrée supplémentaire pour représenter la sélection de toutes les interfaces.
